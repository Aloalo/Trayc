Real time mandelbrot set drawer by Jure Ratkovic.

Controls:
   -Left mouse button + mouse move: pan view
   -Middle mouse button + mouse move: pan view
   -Mouse wheel: zoom in/out
   [q]: zoom in
   [w]: zoom out
   [1]: set low number of iterations
   [2]: set high number of iterations
   [3]: toogle anti aliasing
   [4]: toogle precision
