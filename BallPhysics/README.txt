Particle physics simulator by Jure Ratkovic.

Change initial settings in Settings.ini.

Controls:
   -Left mouse button + mouse move: rotate camera
   -Mouse wheel: zoom in/out
   -p: pause physics
   -f: toogle vector force fields
   -c: toggle concurrency
   -s: toggle solvers
   -LSHIFT: release/bind mouse to the window (for using the gui, exiting ...)
